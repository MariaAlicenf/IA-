# Meu Primeiro Projeto de IA

Este é meu primeiro projeto publicado no GitHub como parte do meu portfólio acadêmico.

## Sobre o projeto
Desenvolvi um chatbot simples em Python com o objetivo de praticar lógica de programação e conceitos introdutórios de Inteligência Artificial.

## Tecnologias utilizadas
- Python

## Funcionalidades
- Interação com o usuário por texto
- Respostas simples baseadas em comandos
- Encerramento do chat com comando "sair"

## Objetivo
Construir meu primeiro projeto para desenvolver minhas habilidades técnicas e iniciar meu portfólio profissional na área de tecnologia.
